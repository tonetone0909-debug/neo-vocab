홈 화면에 추가하면 주소창 없이 진짜 앱처럼 써요. 설치하고 코드로 로그인하면 끝 — 1분이면 돼요!

🔗 tonetone0909-debug.github.io/neo-vocab/
📱 아이폰 SAFARI
  1. 위 주소를 사파리에서 열어요
  2. 아래 공유 버튼(⬆️) 을 눌러요
  3. "홈 화면에 추가" 를 선택해요
🤖 안드로이드 CHROME
  1. 위 주소를 크롬에서 열어요
  2. 오른쪽 위 메뉴(⋮) 를 눌러요
  3. "앱 설치" 또는 "홈 화면에 추가" 를 선택해요
바탕화면에 NT 아이콘이 생겨요.
🔑 코드로 로그인 LOGIN
  1. 받은 코드를 입력하고 들어가기를 눌러요
  2. 코드는 자신의 한글 이름 + 전화번호 뒷 네 자리예요 (예: 홍길동1123)
  3. "로그인 유지" 를 체크하면 다음엔 코드 없이 바로 열려요
코드 입력 → 들어가기
